# ejecutables/

Herramientas locales descargadas (portables, instaladores pequeños).

## Importante

- **No subas** instaladores grandes a GitHub (Ollama, Node, etc.)
- Solo documenta qué instalaste y dónde (`INSTALADOS.md`)
- Archivos `.exe` grandes van en `.gitignore`

## Qué sí guardar aquí

- Scripts launcher `.bat` / `.cmd` que apuntan a rutas locales
- Notas de versiones instaladas

Ver [INSTALADOS.md](INSTALADOS.md)
