# proyectos/

Código fuente de cada aplicación, plugin o sitio.

## Estructura sugerida por proyecto

```
mi-proyecto/
├── AGENTS.md           # Contexto para agentes (Cursor, Cline, Continue)
├── .cursor/rules/      # Reglas del proyecto
├── src/                # Código
└── README.md           # Qué es el proyecto
```

## Ejemplo

```
proyectos/
├── mi-sitio-wordpress/
├── api-rest-node/
└── landing-nextjs/
```

No mezcles varios proyectos en una sola carpeta.
