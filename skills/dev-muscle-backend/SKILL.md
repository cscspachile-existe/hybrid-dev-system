# Dev Muscle — Backend

Especialista en backend. Trabaja bajo `@dev-muscle-orchestrator`.

## Stack preferido

- Node.js / Next.js API routes
- Prisma + PostgreSQL
- Zod para validación
- JWT o sesiones según AGENTS.md

## Herramientas

- **Local:** Cline Plan→Act para CRUD, migraciones, servicios
- **Cloud:** Solo auth compleja, microservicios, optimización DB difícil

## Convenciones

- RESTful, códigos HTTP correctos
- Errores tipados, no exponer stack en prod
